# 🌦️ Weather Forecast Web App

A simple and interactive **7-Day Weather Forecast** web application built using **HTML, CSS, and JavaScript**. This project displays current weather conditions along with a weekly forecast for a selected location.

---

## 📌 Features

* 🌍 Display current location weather
* 🌡️ Shows temperature in Celsius
* 📅 7-day weather forecast
* 🌤️ Weather icons for better visualization
* 🕒 Displays current date
* 🎨 Clean and responsive UI design

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the web page
* **CSS3** – Styling and layout (Responsive design)
* **JavaScript** – Dynamic data handling and API integration

---

## 📂 Project Structure

```
Weather_Forcast-main/
│
├── index.html        # Main HTML file
├── hari.css          # Styling file
├── Script.js         # Main JavaScript logic
├── Script1.js        # Additional JS functionality
```

---

## 🚀 How to Run the Project

1. Download or clone this repository:

   ```
   git clone <your-repo-link>
   ```

2. Navigate to the project folder:

   ```
   cd Weather_Forcast-main
   ```

3. Open `index.html` in your browser.

## 🔧 Configuration

* If this project uses a weather API (like OpenWeatherMap), make sure to:

  * Add your **API key** inside the JavaScript file
  * Replace placeholder values if needed
